各个键盘区之间的距离似乎是半个键位

CX = 15+.5+3+.5+4 = 23
CY = 1+.5+5 = 6.5


假设一个键位为32个像素（SZ），那么
W = SZ*CX = 32*23  = 736
H = SZ*CY = 32*6.5 = 208
